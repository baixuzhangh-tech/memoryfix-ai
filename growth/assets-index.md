## How to quickly assemble a side-by-side image (no editing software)

1. Open Keynote / Google Slides / Figma / free tool `kapwing.com/collage`
2. Drop before-URL on the left half, after-URL on the right half
3. Add a single center divider line (or the words `BEFORE | AFTER` at top)
4. Export at 1600 × 1200 (Reddit compresses anyway)
5. Save as `pair-fleetwood-sidebyside.jpg` etc.

## For short videos

Phone screen recording of the site `/ai-hd` preview flow **is the primary
shot**. Script files specify which pair to run through the live tool — you're
literally screen-recording artgen.site doing its job.
