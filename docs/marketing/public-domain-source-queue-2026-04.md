# Public-Domain Source Queue

Last updated: 2026-04-18

## Purpose

This list is for building the next wave of truthful acquisition cases.

Use these sources as **before images only**. Run them through our pipeline, review the outputs, and only then publish them as product-led case studies.

## Selection Criteria

- strong emotional readability
- family / portrait / couple / parent-child relevance
- public-domain or no-known-restrictions status
- enough texture, damage, or tonal fade to create a visible before/after story
- suitable for `family photo restoration` / `old portrait restoration` / `damaged photo restoration` search intent
